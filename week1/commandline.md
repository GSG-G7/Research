# **Command Line**

## **Defention & Functionality:**

The command line is a text interface for your computer. It’s a program that takes in commands, which it passes on to the computer’s operating system to run.

From the command line, you can navigate through files and folders on your computer, just as you would with Windows Explorer on Windows or Finder on Mac OS. The difference is that the command line is fully text-based.

## **How to use the commands:**

`pwd` => Printing Working Directory

`mkdir` => Make a directory

`touch` => write the file name with its type

`cd` => change directory

`cd ..` => To move up one directory

`cd ../` => To move back one directory

`ls` => List of all files

`rmdir` => remove directory

`rm` => remove file

`rm -rf` => remove all files and directories

`Vim` => to show your code in terminal

`cp` => copy the content from one file to another to have 2 files with the  same content

`mv` => move file from place to another

## **What is (APT) ?**

a free software user interface that handle the installation and removal of software in Linux distro (mint, ubuntu)


## **Why install using package manager ?**
- Update the package repositories
- Update installed software
- Search for available packages
- Get the source code for an installed package
- Reinstall a software package
- Remove a software from your system

## **How to install using package manager ?**
`sudo apt-get install <package_name>`

`sudo apt-get remove <package_name>`

`sudo apt-get purge <package_name>`

`sudo apt-get upgrade`

`sudo apt-get update`
